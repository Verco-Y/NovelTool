# 📄 大荒天道档案馆 - 仙侠人物与剧情线数字化管理系统

## 📌 项目概述

本项目是一款专为大体量、多角色长篇群像小说开发的**数字化创作辅助系统**。
旨在解决宏大叙事作品在长线连载中易出现的「剧情逻辑断层」与「角色战力坍塌」等系统性痛点。项目采用 **AIGC人机协同流** 敏捷构建，通过结构化数据与可视化手段，实现对小说底层世界观与人物设定的强约束管理。

---

## 🛠️ 核心架构与功能说明

### 1. 多维属性矩阵与逻辑拓扑图谱的实时映射

* **设计逻辑：** 将角色的定量属性（六维数值、境界、气运等）与定性的社会关系（阵营、宿命羁绊、敌对状态等）进行解耦与关联。
* **核心功能：** 系统构建了雷达图与网状拓扑图的双向联动机制。当创作者在拓扑图中调整或检索任意角色关系节点时，系统可秒级动态重绘该节点对后续剧情推演的连锁战力影响，在动笔前完成沙盘演算，确保剧情长线发展的系统平衡性。

### 2. 基于状态监听的“异化渲染”与边界防呆机制

* **设计逻辑：** 针对小说中部分偏科角色或特定高潮节点（如觉醒、黑化、道心破碎等突变状态），引入状态机监听机制。
* **核心功能：** 当角色的某项潜能或数值触发预设的设定阈值时，前端系统将执行硬性的“逻辑阻断式提示”——自动重构并渲染独立的“异化主题”（包含色彩反转、噪点动效等）。该机制从视觉体验上强力标记核心伏笔与高能节点，有效防止在高强度日更创作中产生细节遗漏。

---

## 🚀 协同开发特征

* 本项目全流程基于提示词工程（Prompt Engineering）驱动，通过增量式提示（Incremental Prompting）与场景锚定策略，将感性的文本设定需求转化为高精准度的系统指令，实现了技术在泛文学创作场景下的高效提效探索。

---

## 📂 本地部署与运行

1. 双击运行项目根目录下的 `启动.bat` 文件。
2. 开发服务器在后台自动构建完毕后，通过浏览器访问本地相应端口，即可进入交互式可视化剧情推演界面。
